# Installation

To use `auth` in your project, install using `npm`

```sh
npm i @pr4j3sh/auth
```

## Prerequisites

Before using `auth`, ensure that you have the following dependencies set up:

- **Node.js** (version 14 or later recommended)
- **npm** (comes bundled with Node.js)
