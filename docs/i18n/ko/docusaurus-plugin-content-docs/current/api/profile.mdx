```ts
function profile(token: string)
```

<div>
  <span className="badge badge--success">async</span>
</div>

Fetches the authenticated user's profile using the provided token.

**Returns**: Promise&lt;Object&gt;
- The user's profile data.

|PARAMETER|TYPE|OPTIONAL|DEFAULT|DESCRIPTION|
|:---:|:---:|:---:|:---:|:---:|
|token|string|||The JWT access token for authentication.|

```js
import auth from "@pr4j3sh/auth";

const token = "your_jwt_token_here";

const res = await auth.profile(token);
console.log(res);
```
